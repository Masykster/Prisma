âœ¨ About The Project

Prisma is a modern, web-based tool designed for designers, developers, and artists. Built with a focus on UI/UX aesthetics using Glassmorphism principles, Prisma makes color selection an enjoyable experience.

Unlike basic color pickers, Prisma implements complex algorithms to ensure color harmony based on standard color theory rules.

ğŸ’¡ How To Use

1. The Color Wheel Mode

Pick a Color: Drag the cursor on the color wheel canvas.

Select Harmony: Use the dropdown menu to choose a rule (e.g., Complementary).

2. Image Mode

Switch tabs to "From Image".

Drag & Drop an image or click to upload.

Prisma will process the pixel data and generate a palette automatically.
